# Student Advisor Website

A multi-page student advisor platform built with React and Node.js.

## Features

- ✅ Browse expert advisors
- ✅ Book appointments
- ✅ Manage your dashboard
- ✅ Access learning resources
- ✅ Contact support

## Tech Stack

**Frontend:**
- React 18
- React Router
- Axios
- React Icons

**Backend:**
- Node.js
- Express.js
- MongoDB (optional)
- CORS

## Installation

### Prerequisites
- Node.js v14+
- npm or yarn

### Backend Setup

```bash
cd backend
npm install
npm start
